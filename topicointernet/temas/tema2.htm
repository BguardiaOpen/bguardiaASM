Descripción de HTML

HTML (Hyper Text Markup Languaje)

Son documentos texto con extensión .htm o .html

HTML es un 'pseudo-lenguaje' compuesto principalmente de 'tags'. Los tags son como etiquetas que idican una instrucción a ser ejecutada.
 

Carece de estructuras de control tales como ciclos o condicionales y manejo de variables

No es sensitivo a Mayúsculas

Se pueden anidar tags dentro de otros, pero es conveniente anidarlos correctamente

<tag1> <tag2> ... </tag2></tag1>

 

Tags

Los tags poseen la siguiente sintaxis:

<tag parametros=valores> contenido </tag>

Algunos 'tags' sólo abren mientras que existen otros que requieren un tag para abrir y otro para cerrar.

Algunos, además de la característica anterior, utilizan parámetros opcionales.

 

Ejemplos de Tags

Ejemplo de tag que abre y cierra: <b></b>

El tag: <b></b> realiza un formateo de texto y presenta en negritas el texto encerrado en él en el browser.

Ejemplo: "Este texto está en <b>negrita</b>, ¿no es fácil?"

El resultado: "Este texto está en negrita, ¿no es fácil?"

Ejemplo de tag que sólo abre: <hr>

El tag <hr> dibuja una línea horizontal (horizontal rule) en la página Web.

 

Ejemplo de tag con parámetros: <font> </font>
 

El Tag <font></font> utiliza parámetros para modificar las características del texto encerrado, como color o fuente.

<font color="#ff0000">Rojo</font> <font color="#0000ff">Azul</font> <font color="#00ff00">Verde</font>

El resultado: Rojo Azul Verde

 

Estructura de un documento HTML

Estructura general de un documento HTML 

<HTML>
<HEAD>
<TITLE>Mi primer HTML</TITLE> 
</HEAD>
<BODY>Esto es el cuerpo de mi documento</BODY> 
</HTML>
 A la izquierda, vemos un documento HTML básico. Se observa el uso de Tags (etiquetas), las cuales se forman a través de una palabra reservada, la que describe cada elemento del documento, y va encerrada entre los símbolos de mayor y menor. Ejemplos de tags en el documento son: <HTML>, <HEAD>, y <BODY>. 

La mayoría de los Tags van por parejas: uno que abre y uno que cierra; el segundo se señala por el mismo comando o palabra reservada, precedida por una diagonal (slash): / 

Los cuatro tags que hay en este ejemplo son: HTML, que contiene a todo el documento HTML; es decir, todo el texto entre <HTML> y </HTML> es lo que forma propiamente el documento. 

El documento HTML se divide en dos secciones básicas: el encabezado (HEAD) y el cuerpo (BODY). En el encabezado se describe información general del documento, útil para indexarlo. Por ejemplo, el tag TITLE delimita el título del documento, que generalmente aparece en la barra superior del navegador. 

 

Por último, el texto que aparece en la zona principal del navegador, será el que se encuentre entre los tags que abren y cierran BODY. 

HTML no es sensitivo a mayúsculas/minúsculas, ni tampoco a saltos de línea o espacios intermedios.

 

Formateo de texto

<HTML> <HEAD> <TITLE>... </TITLE> <BODY>... </BODY> </HTML>

Los principales tags para formateo de texto son:

<b></b> (bold) El texto encerrado en ellos se presenta en negritas:

   Hola <b>mundo</b> cruel = Hola mundo cruel

<i></i> (italic) El texto encerrado sepresenta italizado

  Hola <i>mundo</i> cruel = Hola mundo cruel

Se pueden anidar los anteriores para presentar texto italizado en bold:

Hola <i><b>mundo</b></i> cruel = Hola mundo cruel

Los tags <i> y <b> pueden ir en diferente orden

<u></u> (underline) presenta texto subrayado

Hola <u>mundo</u> cruel = Hola mundo cruel

Los cambios de línea se hacen a través del tag <br> Los cambios de línea dobles o párrafos se hacen con <p> párrafo </p>  

Se pueden crear encabezados en tamaños predeterminados con <Hx>Encabezado</Hx>

encabezado 1      <H1>encabezado 1 </H1>

encabezado 2                <H2> encabezado 2 </H2>

encabezado 3                          <H3> encabezado 3 </H3>

encabezado 4                                    <H4> encabezado 4 </H4>

encabezado 5                                                       <H5> encabezado 5 </H5>
encabezado 6                                                            <H6> encabezado 6 </H6>

Donde x es un número del 1 al 6 (1 mayor, 6 el menor.)

<center></center> El texto encerrado, se presenta centrado en la pantalla

También se puede conseguir el mismo efecto por medio del tag <p align="center"></p>

  

Inserción de Imágenes

Se utiliza el tag <img>, el cual posee un parámetro obligatorio (src=) y varios opcionales (alt,border,height,width, ismap, align)

Es un tag que no cierra

Sintaxis:

<img src="../../images/386.jpg" width="32" height="32" alt="Chip"> 

El URL (relativo) donde se encuentra el gráfico a presentar es "../images/386.jpg", el cual puede ser formato Compuserve (.GIF) o formato JPEG, que son dos formatos estándares y que permiten reducir en gran medida la transmisión de la información por la red, a través de la compresión.

Las opciones se refieren a:
        a. alt        =
        b. border  = contorno de la imagen
        c. height  =   alto (en cuanto a tamaño)
        d. width   =   ancho (en cuanto a tamaño)
        e. ismap   =
        f. align     =

 

Inserción de Ligas de Hipertexto

El vehículo principal para brincar de una página HTML a otra es la liga de hipertexto

El tag encargado de dicha acción es:

<a href="URL"> texto o imágenes</a>

El URL es la dirección hacia donde se ejecuta la acción:

Ejemplos:

<a href="http://www.sinergia-web.com.mx/documento.html">  ¡Ve a la página de Sinergia!  
<a href="mailto:bguardia@sinergia-web.com.mx">   Mándame un mail....
<a href="ftp://ftp.sinergia-web.com.mx"> ftp://ftp.sinergia-web.com.mx

Lo que encierran los tags se presenta en el browser, subrayado (en texto) o con un margen (imágenes)

Una imagen puede ser también una liga tratándose como "Mapa", en cuyo caso se definen figuras como "hotspots", es decir, secciones de la imagen que funcionan como ligas diferentes. Esto se trata en clase con más detalle pero pueden ver el Ejemplo

Creación de Tablas

En HTML es posible crear tablas por medio del tag <table></table>

<tr></tr> define los renglones

<td></td> las celdas (columnas dentro de cada renglón)

 

Crearemos una tabla de 2 renglones x 3 columnas. A la izquierda se ve el código que la crea, a la derecha cómo queda.

<table border>

<tr><td>1,1</td><td>2,1</td><td>3,1</td></tr>

<tr><td>1,2</td><td>2,2</td><td>3,2</td></tr>

</table>

Creación de Frames

Es posible abrir varias ventanas separadas (con páginas web diferentes) en una misma ventana, se llaman frames.

Se necesita un documento HTML (que carece de <body></body>) que posea la definción del frame

Los tags a utilizarse son:

<frameset></frameset>

<frame>

<noframes>

 

Un ejemplo es la página del Curso, en la parte superior tiene un frame de título y en la inferior otro para el contenido. Si checas el código fuente de esta página, verás que corresponde a los frames.

Utilización de Formularios

El formulario se encierra entre los tags <form> y </form>

No se pueden anidar formularios dentro de otros formularios

El tag <form>, posee como parámetros, el método como se enviará al servidor la información (post/get) y el programa CGI, ASP ó ISAPI al que se le envían los datos (más de esto en la sesión 4)

Si no se indica el método, se tomará por defecto, el método GET.
