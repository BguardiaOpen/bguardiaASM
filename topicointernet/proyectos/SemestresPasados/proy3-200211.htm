Desarrollo en Internet

Proyecto 3: Sistema de Información en Internet.

Equipos de 4 personas.

Entrega: 31 de octubre del 2000

El objetivo del proyecto será proporcionar al usuario un conjunto de servicios a través de formas interactivas e información situada en una base de datos.

Entre las opciones de proyecto, que se sortearán o asignarán entre los equipos durante la clase, se tendrán las siguientes:

  • Sistema de cálculo de impuestos por Internet.
  • Sistema de estadísticas de futbol.
  • Control de inventarios en sucursales.
  • Sistema de boletos para multicines.

 

Sistema de cálculo de impuestos por Internet

Este sistema contará con los módulos que permitirán llenar y calcular los impuestos para honorarios:

  • Actualizar la tabla de impuestos para el trimestre actual.
  • Llenar los ingresos en el trimestre.
  • Llenar las deducciones en el trimestre (sólo las sencillas)
  • Calcular los impuestos de base a pagar.

Para esto, la base de datos contará con las siguientes tablas y campos:

Tabla de "Usuarios"
Nombre del campo Tipo Descripción
RFC Texto RFC único del usuario (13 Caracteres)
CURP Texto Clave única del registro de población (puede estar vacía todavía)
Nombre Texto Nombre del usuario para sus declaraciones

Maximo Moneda Ingresos máximos para que aplique el impuesto

El cálculo del impuesto, en este caso, requerirá los siguientes pasos:

  • El usuario se registra en el sistema.
  • El usuario llena sus ingresos para un trimestre.
  • El usuario llena sus egresos deducibles (se aceptan muy pocos...)
  • El sistema calcula la suma de los ingresos del trimestre.
  • El sistema calcula la suma de los egresos del trimestre.
  • Se calcula la base gravable (ingresos - egresos)
  • Se busca en la tabla de Impuesto a qué rango pertenece de la base gravable, obteniendo de ella la Cuota Fija para el impuesto sobre la renta.
  • Se calcula el Excedente, de acuerdo al rango al que pertenece, como la Base Gravable - el mínimo del rango al que pertenece.
  • Se aplica el Porcentaje sobre el Excedente para calcular el "Impuesto Marginal".
  • Del mismo modo se calcula la Cuota Fija de Subsidio.
  • Se encuentra el Porcentaje de Subsidio Marginal y se aplica sobre el Impuesto Marginal.
  • El ISR final es igual a: Cuota Fija ISR + Impuesto Marginal - Cuota Fija Subsidio - Subsidio Marginal
  • El ISR a pagar es igual al ISR final - el total del ISR retenido
  • El IVA a pagar es igual al total de IVA pagado - el total de IVA Retenido
  • Se presentan todos los datos calculados (para llenar la declaración)

Sistema de estadísticas de futbol

  • Llenar y actualizar información de equipos.

  • Registrar resultados de los partidos: marcador y anotadores de los goles.

Control de inventario en sucursales

  • Registrar el catálogo de productos.
  • Llevar el catálogo de sucursales.
  • Registrar ventas y reducciones de inventario de una sucursal.
  • Registrar adquisiciones y subida en inventario a una sucursal.

Tabla de "Sucursales"
Nombre del campo Tipo Descripción
IDSucursal Autonumérico Consecutivo para indicar el número de sucursal
NombreSucursal Texto Nombre por el que se conoce la sucursal

Nombre del campo Tipo Descripción

IDSucursal Texto Descripción del producto
Cantidad Entero largo Cuánto hay del producto en la sucursal
